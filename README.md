# Rotaciones Clínicas - Universidad EIA

Sitio web informativo sobre el programa de rotaciones clínicas del internado de la Universidad EIA en el Hospital Pablo Tobón Uribe.

## 📁 Estructura del Proyecto

```
rotaciones/
├── index.html          # Estructura HTML limpia y semántica
├── styles.css          # Estilos CSS separados y organizados
├── script.js           # Lógica JavaScript modular
├── image.png           # Imagen hero de la Universidad EIA
├── images.jpeg         # Imagen del Hospital Pablo Tobón Uribe
└── README.md          # Este archivo
```

## ✨ Características

- **Diseño responsivo**: Adaptado a dispositivos móviles, tablets y desktops
- **Interfaz limpia**: Paleta de colores corporativa (azul marino, oro)
- **Contenido completo**: Sin truncamientos, información íntegra
- **Código separado**: HTML, CSS y JavaScript en archivos independientes
- **Accesibilidad mejorada**: Meta tags, alt text descriptivos, navegación clara
- **Meta tags SEO**: Descripción y viewport configurados correctamente

## 🔧 Cómo usar

### Instalación local
```bash
git clone https://github.com/franco910201/rotaciones.git
cd rotaciones
```

No requiere instalación de dependencias. Solo abre `index.html` en tu navegador.

### Estructura de archivos

#### `index.html`
- Código HTML semántico y limpio
- Secciones bien organizadas
- Links a archivos externos (CSS y JS)
- Atributos `alt` descriptivos en imágenes

#### `styles.css`
- Variables CSS para fácil personalización
- Grid responsivo con `auto-fit`
- Media queries para mobile, tablet y desktop
- Estilos para componentes reutilizables

#### `script.js`
- Función `switchTab()` para navegación entre secciones
- Event listeners optimizados
- Comentarios JSDoc
- Validaciones de elementos

## 📱 Responsividad

El sitio se adapta automáticamente a:
- **Desktop**: Grilla de 2 columnas (450px mín)
- **Tablet**: Grilla de 1 columna (768px)
- **Mobile**: Diseño vertical optimizado (480px)
